Sistema de Lutadores – Java
Este projeto foi desenvolvido como um exercício de Programação Orientada a Objetos em Java.
O objetivo é criar uma classe que represente um lutador e permita controlar suas informações e histórico de lutas.

Descrição do Projeto
O sistema possui uma classe chamada Lutadores, que armazena dados como:

Nome
Nacionalidade
Idade
Altura
Peso
Categoria
Número de vitórias
Número de derrotas
Número de empates

Também possui métodos para apresentar o lutador, mostrar seu status e atualizar os resultados das lutas.

Funcionalidades

Exibir os dados do lutador
Mostrar o histórico de lutas
Registrar vitória, derrota ou empate
Definir a categoria do lutador de acordo com o peso


Tecnologias Utilizadas

Java
Programação Orientada a Objetos (POO)


Finalidade Acadêmica
Projeto desenvolvido para fins acadêmicos, com o objetivo de praticar:

Criação de classes e objetos
Uso de construtores
Encapsulamento
Métodos getters e setters
Estruturas condicionais
